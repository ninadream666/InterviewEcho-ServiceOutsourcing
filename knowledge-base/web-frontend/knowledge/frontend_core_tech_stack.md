# 前端面试核心技术栈考纲 (Frontend Core Tech Stack)

1. HTML与Web标准：熟练掌握HTML5语义化标签、文档类型声明(DOCTYPE)、meta标签及SEO优化机制，理解src与href的本质区别。
2. 元素与布局机制：深刻理解行内元素与块级元素的特性及转换，掌握页面基本流与文档树结构。
3. 资源加载与性能：清楚link与@import加载CSS的区别、页面渲染模式（标准与怪异模式）及常见图片格式（GIF/JPG/PNG等）的适用场景。
4. HTML5新特性：熟练使用H5新增的选择器、WebStorage（localStorage/sessionStorage）、WebWorker、WebSocket、Canvas/SVG及表单增强。
5. JS基础与DOM：掌握JavaScript的基础类型、闭包、原型链、跨域请求原理，了解DOM操作防抖与节流、事件轮询、垃圾回收机制。
6. 前端框架生态：深入理解Vue/React的响应式原理、虚拟DOM机制、Diff算法及生命周期钩子流转。
7. 计算机网络：熟悉TCP/IP、HTTP/HTTPS、资源缓存策略（强缓存与协商缓存）、跨域处理策略及Web安全防范（XSS/CSRF）。
8. 工程化与打包：掌握Webpack/Vite等构建工具的核心插件原理、Loader机制及性能调优方案。
